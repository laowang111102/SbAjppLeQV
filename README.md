# 个人记账管理系统 python667

## 项目概述

个人记账管理系统是一款帮助用户便捷管理个人财务的应用。系统提供登录、个人信息管理、收入与支出记录以及统计等功能，使用户能够高效地记录和分析个人财务状况。

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：HTML

## 功能模块

### 用户模块
- 用户登录
- 个人信息修改

### 记账管理模块
- 收入记录
- 支出记录
- 收入记账管理
- 支出记账管理

### 统计模块
- 收入统计
- 支出统计

## 系统角色

- 普通用户：登录系统，管理个人财务信息。

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x

## 部署步骤

1. 安装Python环境和Django框架。
2. 创建MySQL数据库和表结构。
3. 配置应用环境。
4. 运行Django应用。

## 核心亮点

- **简单易用**：用户界面友好，操作便捷。
- **个性化记账**：支持自定义收入和支出类别，满足不同用户的需求。
- **数据分析**：提供收支统计功能，帮助用户更好地理解和管理个人财务状况。
